## Cos'è AutoGLM Phone 9B?

*Scritto da Matteo Giardino* - CTO e Founder di OpenClaw.

AutoGLM Phone 9B è un modello linguistico multimodale da 9 miliardi di parametri ottimizzato specificamente per comprendere e interagire con le interfacce grafiche mobile. A differenza degli LLM general-purpose, è stato addestrato su enormi quantità di registrazioni di schermi Android.

## Come Funziona Sotto il Cofano

L'architettura si basa su un ciclo continuo di osservazione e predizione:
1. **Percezione**: Il modello riceve uno screenshot dello stato corrente di Android.
2. **Analisi**: Identifica gli elementi UI azionabili.
3. **Predizione dell'Azione**: Basandosi sull'obiettivo in linguaggio naturale, genera le coordinate esatte e il tipo di azione da eseguire.

## Guida al Setup Locale

Eseguire un modello multimodale da 9B richiede un hardware adeguato. Avrai bisogno di un Mac con 16GB di RAM unificata o di un PC con RTX 3090/4090.

Ecco come inizializzare l'agente:

```bash
# Clona il repository e prepara l'ambiente
git clone https://github.com/THUDM/AutoGLM.git
cd AutoGLM
pip install -r requirements.txt

# Avvia il server di inferenza locale
python server.py --model THUDM/AutoGLM-Phone-9B
```

Una volta avviato il server, puoi collegare il tuo dispositivo Android tramite ADB e lanciare lo script per avviare l'automazione.
