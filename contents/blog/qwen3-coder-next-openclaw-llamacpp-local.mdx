## Installing llama.cpp
First, install llama.cpp to run the model efficiently on your hardware.
```bash
git clone https://github.com/ggerganov/llama.cpp
cd llama.cpp
make
```

## Configuring OpenClaw
OpenClaw is the framework that will orchestrate the model. Connect it to the local llama.cpp server in your config.

## Running locally
Start the server and enjoy your private AI coding assistant!
