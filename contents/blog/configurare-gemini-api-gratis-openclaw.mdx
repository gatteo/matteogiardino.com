Usare agenti AI in locale è fantastico, ma i modelli migliori spesso richiedono hardware costoso o API a pagamento. L'API di Google Gemini offre un generoso piano gratuito (15 richieste al minuto), perfetto per alimentare gli agenti OpenClaw senza spendere un centesimo. Dopo aver configurato i miei agenti su questa infrastruttura, ti spiego esattamente come impostare le chiavi e i modelli nel tuo file di configurazione.

## Perché usare il piano gratuito di Gemini con OpenClaw

L'API gratuita di Gemini (inclusi i modelli 3.1 Pro e Flash) è l'ideale per sviluppatori che vogliono testare flussi di lavoro complessi senza svuotare il portafoglio. A differenza dell'API di OpenAI, che richiede crediti prepagati, il livello gratuito di Google Studio AI è sempre attivo.

Il vantaggio principale è la finestra di contesto enorme (fino a 2 milioni di token) che ti permette di dare in pasto all'agente intere basi di codice, senza preoccuparti troppo dell'ottimizzazione dei prompt.

## Come ottenere la tua chiave API Gemini gratis

### 1. Accedi a Google AI Studio

Il primo passo è creare una chiave. Vai su [Google AI Studio](https://aistudio.google.com/) e accedi con il tuo account Google. Non è necessario inserire una carta di credito per il piano gratuito.

### 2. Genera la chiave

Clicca su "Get API key" e poi su "Create API key". Copia la stringa alfanumerica generata. Conservala in un posto sicuro, perché ne avrai bisogno per il file di configurazione di OpenClaw.

## Configurare OpenClaw per usare Gemini

Ora che hai la chiave, dobbiamo dire a OpenClaw di usarla. Apri il file di configurazione principale di OpenClaw. Di default si trova in `~/.openclaw/openclaw.json`.

Devi aggiungere la chiave sotto l'oggetto `env` e impostare Gemini come modello principale per i tuoi agenti.

```json
{
  "env": {
    "GEMINI_API_KEY": "LA_TUA_CHIAVE_QUI",
    "GEMINI_MODEL_TIMEOUT_SECONDS": "600"
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "google/gemini-3.1-pro-preview"
      }
    }
  }
}
```

Salva il file. OpenClaw ricaricherà automaticamente la configurazione, o potrai forzarlo riavviando il gateway.

## Limitazioni e workaround (Il problema delle quote)

C'è un inghippo: il piano gratuito è limitato a 15 RPM (richieste al minuto) e 1500 RPD (al giorno). Se il tuo agente OpenClaw fa troppi loop veloci, andrai incontro a errori di rate limiting (Quota Exceeded).

Una soluzione pratica che ho adottato è aumentare i delay tra le azioni dell'agente o combinare Gemini con modelli locali leggeri. Se vuoi approfondire, puoi leggere come gestisco le [quote di Antigravity con Gemini 3.1 Pro](/blog/agent-manager-antigravity-guida).

## Domande frequenti

### Quanto costa usare l'API di Gemini?

Il piano base per gli sviluppatori su Google AI Studio è completamente gratuito e offre accesso a Gemini 1.5 Flash e Gemini 1.5 Pro, con limiti di 15 richieste al minuto. Non serve carta di credito.

### Posso usare Gemini gratis per scopi commerciali?

I termini di servizio del piano gratuito di Google permettono l'uso per prototipazione e sviluppo, ma attenzione: i dati inviati tramite l'API gratuita potrebbero essere utilizzati da Google per addestrare i propri modelli. Se lavori con dati sensibili, valuta [soluzioni open source in locale](/blog/configurare-openclaw-senza-api-key-gratis).

### Cosa succede se supero le 15 richieste al minuto?

L'API restituirà un errore HTTP 429 (Too Many Requests). OpenClaw, se ben configurato con le retry policies, aspetterà qualche secondo prima di ritentare.

## Conclusione

Configurare Gemini su OpenClaw è il modo più rapido ed economico per accedere a un modello AI di classe "frontier" senza costi operativi. Con 15 richieste al minuto, hai tutto il margine per testare agenti complessi. Assicurati solo di monitorare le quote se decidi di scalare in produzione.
