# Come Installare e Usare Claude Code in Antigravity

Claude Code è un assistente AI eccezionale, ma per farlo funzionare bene dentro Antigravity servono alcuni passaggi specifici. In questa guida ti mostrerò come configurarlo.

## Cos'è Claude Code?

Claude Code è l'assistente alla programmazione CLI di Anthropic. Ti permette di eseguire task complessi direttamente dal terminale.

<CtaCard title="Iscriviti alla Newsletter" description="Ricevi gli ultimi tutorial su AI e OpenClaw direttamente nella tua casella di posta." url="/newsletter" />

## Perché usarlo in Antigravity?

Antigravity fornisce un ambiente specializzato per gli agenti AI. Combinandolo con Claude Code, puoi collegarlo senza problemi a OpenClaw.

## Passaggi per l'installazione

1. Assicurati di avere l'ultima versione della CLI di Antigravity.
2. Esegui `npm install -g @anthropic-ai/claude-code`.
3. Autenticati con la tua chiave API Anthropic.

## Collegare Claude Code a OpenClaw

Una volta installato, puoi esporre gli strumenti MCP di OpenClaw a Claude Code per farli collaborare su refactoring complessi del codice.
