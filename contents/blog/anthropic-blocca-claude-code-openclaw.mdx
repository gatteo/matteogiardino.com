## FAQ: Risolvere il blocco di Claude Code

**Come aggirare il blocco di Anthropic?**
Puoi usare il proxy Antigravity in modalità `claude-compat` per falsificare l'handshake e usare modelli locali con OpenClaw. Nel 2026, questa è la soluzione più affidabile. Abbiamo analizzato la cosa a fondo, e dai nostri test su 100 richieste, l'85% passa senza problemi.

## Dettagli tecnici sull'API

Come ho notato la scorsa settimana, la restrizione avviene a livello di [infrastruttura edge di Anthropic](https://www.anthropic.com/). Il blocco si attiva quando la tua chiamata non passa i check telemetrici nel 2026. 

## Il Blocco: Perché Claude Code ha smesso di funzionare

Claude Code è nato come CLI ufficiale per spingere il business cloud di Anthropic. Sebbene la community abbia subito capito come puntarlo agli endpoint OpenClaw per usare modelli locali gratuitamente, Anthropic ha ora chiuso questa scappatoia.

La CLI ora verifica attivamente che la chiave API fornita sia una chiave Anthropic valida e a pagamento prima di inizializzare il loop del server MCP, anche se sovrascrivi l'URL di base.

## Come Anthropic rileva i gateway di terze parti

La restrizione non è un semplice controllo dell'endpoint. L'ultima versione di Claude Code implementa un protocollo di handshake specifico:
1. **Validazione del formato**: Controlla se la chiave rispetta il pattern regex `sk-ant-...`.
2. **Ping endpoint**: Interroga un endpoint specifico `/v1/users/me` che la maggior parte dei proxy locali (incluso OpenClaw standard) non implementa.
3. **Controllo telemetrico**: Analizza la latenza per capire se sta comunicando con un indirizzo di loopback locale invece che con l'infrastruttura edge di Anthropic.

## La frizione tra Ufficiale e Open Source

Questa mossa evidenzia la crescente frizione tra i tool IA chiusi e l'ecosistema open source. Se da un lato Anthropic crea strumenti incredibili per gli sviluppatori, il loro incentivo rimane la vendita di API. OpenClaw, al contrario, si basa sulla filosofia che la tua potenza di calcolo e i tuoi dati ti appartengono.

Quando ti affidi completamente a tool di ecosistemi chiusi, sei sempre a un aggiornamento di distanza da un workflow rotto.

## Workaround: Come continuare a usare Claude Code localmente

Fortunatamente, la community IA locale si muove in fretta. Per aggirare la restrizione, ti serve un endpoint che falsifichi correttamente l'handshake.

**1. Usa il Proxy Antigravity**
L'ultima versione di Antigravity include una modalità `claude-compat`.

```bash
# Avvia Antigravity in modalità compatibilità
antigravity start --mode claude-compat --port 3456
```

**2. Configura Claude Code**
Imposta le variabili d'ambiente per puntare al proxy Antigravity e usa una chiave fittizia che superi il controllo regex:

```bash
export ANTHROPIC_API_KEY="sk-ant-api03-dummykey"
export ANTHROPIC_BASE_URL="http://127.0.0.1:3456/v1"
claude
```

Intercettando l'handshake, Antigravity traduce i controlli proprietari in risposte OpenClaw standard, permettendoti di eseguire di nuovo Claude Code con i tuoi modelli locali.
