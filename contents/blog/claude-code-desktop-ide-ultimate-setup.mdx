## Setting Up Your Environment

### 1. Installation

First, you need to install the desktop application. Ensure you have the latest prerequisites.

```bash
npm install -g @anthropic-ai/claude-code
```

### 2. Configuring Ollama Integration

One of the best features is routing the Desktop IDE through Ollama to use local models like Llama 3 or Qwen without paying for API keys.

```bash
claude config set backend ollama
claude config set model qwen-2.5-coder
```

## Common Gotchas

When first connecting to local servers, you might encounter a CORS or port binding issue if Ollama is running on a non-standard port. Make sure to export `OLLAMA_HOST=0.0.0.0:11434` in your profile.

## FAQ

### Does Claude Code work entirely offline?

Yes, if you configure it to use Ollama models locally, it requires zero internet connection to generate code.

### Is the Desktop IDE free?

The software itself is free. You only pay for API usage if you choose to route through Anthropic's cloud instead of a local model.

### How does it compare to OpenClaw?

While OpenClaw is amazing for multi-agent workflows and task delegation, Claude Code Desktop provides a more native developer experience for single-agent codebase operations.
