## Configurare l'Ambiente

### 1. Installazione

Prima di tutto, devi installare l'applicazione desktop. Assicurati di avere gli ultimi prerequisiti.

```bash
npm install -g @anthropic-ai/claude-code
```

### 2. Integrazione con Ollama

Una delle funzionalità migliori è indirizzare il Desktop IDE tramite Ollama per utilizzare modelli locali come Llama 3 o Qwen senza pagare per le API key.

```bash
claude config set backend ollama
claude config set model qwen-2.5-coder
```

## Problemi Comuni

Durante la prima connessione ai server locali, potresti incontrare problemi di CORS o di binding delle porte se Ollama è in esecuzione su una porta non standard. Assicurati di esportare `OLLAMA_HOST=0.0.0.0:11434` nel tuo profilo.

## Domande Frequenti

### Claude Code funziona completamente offline?

Sì, se lo configuri per usare i modelli Ollama in locale, non richiede alcuna connessione internet per generare codice.

### Il Desktop IDE è gratuito?

Il software in sé è gratuito. Paghi solo per l'utilizzo delle API se scegli di instradare tramite il cloud di Anthropic invece di un modello locale.

### Come si confronta con OpenClaw?

Mentre OpenClaw è fantastico per workflow multi-agente e deleghe di task, Claude Code Desktop offre un'esperienza per sviluppatori più nativa per le operazioni sul codice a singolo agente.
