## Installare llama.cpp
Per prima cosa, installa llama.cpp per eseguire il modello in modo efficiente sul tuo hardware.
```bash
git clone https://github.com/ggerganov/llama.cpp
cd llama.cpp
make
```

## Configurare OpenClaw
OpenClaw è il framework che orchestrerà il modello. Collegalo al server locale llama.cpp nella tua configurazione.

## Esecuzione in locale
Avvia il server e goditi il tuo assistente AI privato per il codice!
